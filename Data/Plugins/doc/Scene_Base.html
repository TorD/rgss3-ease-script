** Scene_Base EXTENSION

Version: 1.0.1 Date: 08/12/2013 Author: Galenmereth / Tor Damian Design

Extended for: TDD Easing Script

This calls update on the new Ease module, so that all applied easings are updated each frame tick. Nothing else.
